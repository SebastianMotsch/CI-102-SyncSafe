Introductions

Public Wi-Fi offers convenience but often comes with serious security risks.
Understanding the various security protocols and the vulnerabilities associated with them is key to protecting personal data when connecting to these networks.
This page provides an overview of common security terms and potential threats faced by users on public Wi-Fi.

Important Facts

Older Protocols are Weak:

WEP, once widely used, is now obsolete due to its use of a single, static encryption key that is easy to crack.
WPA, though more secure, can still be compromised through brute-force attacks.

Weak Passwords are Risky:

Even modern protocols like WPA2 can be vulnerable if users create weak passwords,
allowing hackers to perform dictionary attacks.
